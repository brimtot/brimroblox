## ⚡ Overview

**BrimBox** is a lightweight, feature-rich Lua script designed for modern Roblox executors (like Xeno). It features a sleek, draggable glassmorphism UI and a powerful suite of physics and visual modifications. 

No keys, no ads, no backend servers—just execute and play.

> ⚠️ **DISCLAIMER (DWYOR):** This script interacts with game memory and physics. **Do With Your Own Risk.** Using this in public servers can result in an account ban. 

---

## 🚀 Features

- 👁️ **Recon (ESP):** Highlights targets (differentiates bots vs. real players).
- 🏃 **Speed:** Configurable movement velocity multiplier.
- 👻 **Noclip:** Physics-based collision bypass with adjustable speed.
- 🥔 **Potato FX:** Removes textures, shadows, and particles for massive FPS boosts.
- 🎯 **Hitbox Expander:** Configurable X, Y, Z hitbox scaling.
- 🧹 **MapClear:** Dynamically hides unprotected map clutter.
- 🐒 **Monkey Mode:** Physics-based vertical wall climbing.
- ☀️ **FullBright:** Overrides game lighting and removes fog/atmosphere.
- ⚡ **Insta-Interact:** Removes proximity prompt hold durations (E to interact instantly).

## 🕹️ How to Use

### Execution
You can run BrimBox using any standard Roblox executor (Xeno, Fluxus, Delta, etc.). Paste the following code into your executor and hit **Execute**:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/brimtot/brimroblox/refs/heads/main/file/script.lua"))()
```

(Optional: You can also place the script inside your executor's autoexec folder to load it automatically every time you join a game).

Controls & Keybinds
You can toggle all features directly from the UI, or use the following hotkeys for quick access.

Hold ALT + the following keys:

ALT + 1 : Toggle Recon

ALT + 2 : Toggle Speed

ALT + 3 : Toggle Noclip

ALT + 4 : Toggle Potato FX

ALT + 5 : Toggle Hitbox

ALT + 6 : Toggle MapClear

ALT + 7 : Toggle Monkey Mode

ALT + 8 : Toggle FullBright

ALT + 9 : Toggle Insta-Interact

ALT + S : Show/Hide Menu
